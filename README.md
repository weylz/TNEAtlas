# 🧬 TNEAtlas: A Pan-Cancer Database for Transcribed Non-Coding Elements

**TNEAtlas** is the first systematic research database focused on **Transcribed Non-Coding Elements (TNEs)** in cancer genomes. It integrates multi-omics data to provide a one-stop solution for exploring the regulatory mechanisms of non-coding elements in tumorigenesis and development. It facilitates insights into the role of functional non-coding elements in gene regulation and cancer biology, and assists in the discovery of therapeutic targets and biomarkers.
